# Mini Chat System (Python)

## Overview
Mini Chat System is a Python-based application that enables registered users to exchange messages in a simple and structured way. The system supports user authentication, inbox management, and persistent message storage using JSON files.

## Features
- User registration and login
- Send messages between registered users
- User-specific inbox to view received messages
- Validation to prevent invalid actions (e.g., self-messaging)
- Persistent data storage using JSON

## Tech Stack
- Python
- JSON File Handling
- Object-Oriented Programming (OOP)

## How It Works
1. Users sign up or log in to the system
2. Authenticated users can send messages to other registered users
3. Messages are stored persistently and retrieved per user
4. Each user can view their own inbox securely
